# TicTacToe (Swift)
Just a simple iOS app to hammer down on some fundamentals.

Concepts utilized:
* UIStackView (Vertical and Horizontal)
* UIButton
* Dispatch Queue
* UIAlert
* MVC Pattern
* While loop
* Conditionals (If/Else)
* Control Flow (Case/Switch)
* Functions
* Classes
* Static variables/functions
* Array
